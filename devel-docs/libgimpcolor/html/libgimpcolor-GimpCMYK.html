gimp_cmyk_set ()

void                gimp_cmyk_set                       (GimpCMYK *cmyk,
                                                         gdouble cyan,
                                                         gdouble magenta,
                                                         gdouble yellow,
                                                         gdouble black);

Very basic initialiser for the internal GimpCMYK structure. Channel values are doubles in the range 0 to 1.
